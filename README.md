# docker-try-out
A simple project to understand docker

## Start/Restart all containers
`python start.py`
## Server addresses
Container | Address
------------ | -------------
logging | `localhost:8080`
static | `localhost:8081`
## Access containers
Container | Command
------------ | -------------
master | `master/access.sh`
logging | `logging/access.sh`
static | `static/access.sh`
